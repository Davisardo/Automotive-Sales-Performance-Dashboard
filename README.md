## Project Overview  
This project presents an interactive **Power BI dashboard** designed to analyze **automotive sales performance** across products, dealers, and regions.  
The dashboard provides a comprehensive view of sales trends and key business metrics to support **data-driven decision making**.

## Problem Statement  
**The objective of this project is to monitor and evaluate overall automotive sales performance by analyzing product sales, dealer performance, and regional distribution.**  
This helps businesses identify growth opportunities, optimize sales strategies, and improve overall performance.

## Data Used  
- **Data** – Automotive sales dataset containing product, dealer, and regional sales information  
- **Data Preparation** – Microsoft Excel  
- **Data Modeling & DAX Functions** – Power BI  
- **Data Visualization & Dashboard** – Power BI  

## KPI’s Requirement  
1. **Total Revenue** – Measure overall sales performance.  
2. **Units Sold** – Track total quantity of products sold.  
3. **Revenue by Region** – Analyze contribution from each region.  
4. **Revenue by Dealer** – Evaluate dealer performance.  
5. **Product Performance** – Identify top and low-performing products.  

## Charts Requirement  
1. **Revenue by Region (Bar Chart):** Visualize regional sales contribution.  
2. **Units Sold by Dealer (Bar Chart):** Compare dealer performance.  
3. **Revenue Trend Over Time (Line Chart):** Analyze sales trends.  
4. **Top Products by Revenue (Bar Chart):** Highlight best-performing products.  
5. **Sales Distribution (Pie/Donut Chart):** Show proportional contribution.  

## Key Features  
- Modern, clean, and professional **UI design**  
- **Interactive KPI cards** for quick performance insights  
- **Dynamic data visualization** including bar charts, line charts, and regional analysis  
- **User-friendly layout** for better data exploration  
- Fully interactive dashboard for sales and performance monitoring  

## Outcomes  
**This dashboard enables businesses to:**  
- Monitor overall sales performance across multiple dimensions  
- Identify top-performing dealers and products  
- Understand regional sales distribution and trends  
- Support strategic decision-making using data-driven insights
